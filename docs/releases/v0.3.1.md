# QueryAssure v0.3.1 — public launch and verified distribution

QueryAssure makes SQL Agent behavior testable before it reaches production. Version 0.3.1
turns the project into a complete public release: a zero-key hosted playground, reusable
quality-gate action, verified Python artifacts, container images, checksums, contribution
paths, and security guidance.

## Highlights

- **Experience the whole loop:** ask Northstar Retail questions, inspect metadata retrieval,
  generated SQL, policy decisions, results, and the evaluation suite in the hosted playground.
- **Test any compatible agent:** use a Python callable or HTTP endpoint without adopting a
  required orchestration framework.
- **Block unsafe regressions:** validate parsing, read-only behavior, schema grounding,
  sensitive columns, result equivalence, latency, and tool-call budgets in CI.
- **Ground with real metadata:** import PostgreSQL schemas and dbt manifests while keeping
  credentials out of generated catalogs.
- **Reproduce the evidence:** deterministic data, fixed seeds, executable gold queries,
  JSON reports, correctness-first leaderboards, and SHA-256 checksums.

## Install

```bash
pip install https://github.com/Victoria824/QueryAssure/releases/download/v0.3.1/queryassure-0.3.1-py3-none-any.whl
queryassure --version
queryassure seed --orders 1000
queryassure test --suite evals/retail.yml
```

Container images produced by the release workflow:

```text
ghcr.io/victoria824/queryassure-api:0.3.1
ghcr.io/victoria824/queryassure-playground:0.3.1
```
